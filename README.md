# miniProgramForm
小程序表单小工具，可上传缩略图，文字，多图等内容，可自定义编辑

## 效果展示
![image](https://github.com/939682389/miniProgramForm/blob/master/QQ20201001-153000%402x.png)

## 用法
```bash
  "usingComponents": {
    "tool": "../../component/tool/tool"
  }
```

## 返回格式
```
{
  title: "title", 
  contents: : [
    {type: "文字", content: "获取↵"},
    {type: "大图", content: Array(1)}
    ]
}
```
